How to Cancel Amazon Try Before You Buy

Easy — CFS 1/5
Method: online · Last verified: 2026-04-13 · Official site

Cancellation Steps

  1. Amazon Try Before You Buy is not a subscription — it's a free Prime member feature. To stop using it, simply don't order more try-on boxes.
  2. For active try-on orders: return unwanted items within 7 days using the prepaid return label. You are only charged for items you keep. If you fail to return items within 7 days, you will be charged the full price.
  3. To manage or cancel pending Try Before You Buy orders, go to Your Orders on amazon.com. There is no recurring subscription to cancel — this is a per-order service.

⚠ Watch Out — Dark Patterns

⚠ auto-charge deadline

If you don't return items within the 7-day try-on window, you're automatically charged full price. The deadline can be easy to miss, especially for busy shoppers.
